# Ordinacija West - Web Sajt

Moderni i profesionalni web sajt za privatnu ordinaciju West doktora Tamare Vesović.

## 📋 Opis sajta

Kompletna web prezentacija stomatološke i estetske ordinacije sa:
- **Responsive dizajn** (desktop, tablet, mobile)
- **Modern medical design** (bela, bež, svetlo plava paleta)
- **SEO optimizovana struktura**
- **Kontakt forma sa validacijom**
- **Galerija sa filter funkcionalnostью**
- **Smooth scroll navigacija**
- **Mobile hamburger menu**

## 📁 Struktura fajlova

```
├── index.html          # HTML struktura (semantic, SEO optimizovan)
├── style.css           # CSS stilovi (responsive, animations)
├── script.js           # JavaScript funkcionalnost
└── README.md           # Ovaj fajl
```

## ✅ Implementirano

### Sekcije:
- **Header** - sticky navigacija, logo, dropdown podmenu, hamburger menu
- **Hero** - naslov, CTA dugmići, hero slika
- **O nama** - tekst o doktoru, feature cards
- **Usluge** - stomatologija i estetska medicina
- **Galerija** - filter, modal prikaz slika
- **Recenzije** - prikaz recenzija pacijenata sa zvezdama
- **FAQ** - accordion česta pitanja
- **Lokacija** - info o ordinaciji, mapa placeholder
- **Kontakt** - forma sa validacijom, kontakt info
- **Footer** - linkovi, social media, copyright

### Funkcionalnosti:
- Potpuna responsivnost (mobile-first)
- **Dropdown podmenu** u navigaciji
- Smooth scroll između sekcija
- Form validacija (real-time)
- Gallery filtering
- Image modal
- **FAQ accordion** funkcionalnost
- **Review cards** sa zvezdama
- Scroll animacije
- Back-to-top dugme
