# Slike za Recenzije

Ovde stavite slike ljudi za recenzije. Svaka slika treba da bude:

## Format slika:
- Kvadratne (1:1 omjer)
- Minimum 120x120 piksela
- JPG ili PNG format
- Manje od 100KB
- Dobro osvetljene, jasne slike lica

## Nazivi fajlova:

### Ženske recenzije:
- `marija-p.jpg` - Marija P. (fiksne proteze)
- `stefana-m.jpg` - Stefana M. (mezoterapija)
- `ana-j.jpg` - Ana J. (botox)
- `jelena-s.jpg` - Jelena S. (biorevitalizacija)
- `milena-r.jpg` - Milena R. (fileri)
- `milica-t.jpg` - Milica T. (hijaluron)

### Muške recenzije:
- `petar-n.jpg` - Petar N. (fiksne proteze)
- `nenad-v.jpg` - Nenad V. (fiksne proteze)

## Kada budete imali slike:
1. Stavite ih u ovaj folder
2. Pozovite me da ažuriram HTML kod
3. Automatski će zameniti ikone sa slikama

## Trenutno stanje:
- Koriste se ikone umesto slika
- CSS je spreman za slike
- Kada dodate slike, samo treba ažurirati HTML
